## Getting Started

1. `npm install`
2. Grab your API key from OpeanSea
3. Create a `.env.local` file and paste the key
4. `npm run dev` or `yarn dev`
   
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
